## 케이스 주제

랜덤의 수를 만들어서, 배열의 인덱스에 대입하기
결과값을 이어붙히기

## 기능 요구사항

1. 버튼을 누를 시, 주어진 배열을 참조해 랜덤으로 hex코드가 생성된다
2. 생성된 랜덤 hex 코드로 body의 bg를 바꾼다
3. span에 텍스트가 출력된다
